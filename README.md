# FoDS Project

Group project repository.

## Project structure

- `data/` dataset files
- `notebooks/` exploratory notebooks
- `src/p1/` preprocessing code
- `src/p2/` statistical testing
- `src/p3/` machine learning models and evaluation
- `src/p4/` feature importance and related analysis
- `figures/` exported plots
- `report/` written report sections

## Branch workflow

- `main` is the stable shared branch
- each team member creates a separate branch from `main`
- finished work gets merged back into `main`
